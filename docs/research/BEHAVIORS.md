# YUYU Full Site Behaviors

## Explore

- Category tabs switch prompt prefix and template cards.
- Template cards populate the prompt and open the workflow panel.
- Prompt tool buttons open contextual panels; chips append text to the prompt.
- Model selector, upload, mention, style, notifications, invite, preview, and export actions use the shared modal system.
- Send action advances a four-step generation workflow and decrements credits.

## Project Center

- Search filters project cards.
- New project prepends a draft project.
- Duplicate creates a copy of the selected card.
- Continue opens the canvas editor.
- Export opens an export modal and confirmation decrements credits.

## Asset Center

- Asset tabs render separate mock datasets.
- Clicking an asset marks it selected.
- Add-to-current-project opens the workflow panel at the asset step.

## Team And Account

- Comments are inserted into the feed locally.
- Invite opens a permission modal.
- Approval and notification actions provide toast feedback.
- Account credits stay in sync with the side navigation.

## Canvas Editor

- Add node creates a new selected node.
- Nodes are draggable with pointer input.
- Zoom controls update the canvas plane and visible zoom label.
- Auto layout restores default node positions and zoom.
- Inspector edits update the selected node.
- Preview and export open shared modals.
